## Fortran notes

### Installation + Setup

Very important to do these steps in order!

1. Setup Anaconda environment:

```yml
name: my_fortran_project
channels:
        - conda-forge
dependencies:        
        - pip
        - fortls
        - findent
```

2. Install VSCode `modern-fortran` extension
